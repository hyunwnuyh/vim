# hyunwnuyh's NeoVim settings

## How to install
1. Install NeoVim
2. Download git repository
3. go to cloned repository folder and give permission to install.sh
4. run install.sh
----------------------------------------------------
    git clone https://github.com/hyunwnuyh/vim.git
    cd vim
    chmod 777 install.sh
    sudo ./install.sh
----------------------------------------------------
